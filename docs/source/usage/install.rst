Install
-------

在终端中运行以下命令来安装项目：

从PyPI上安装最新版本::

    pip install snngrow

从GitHub上安装::

    git clone https://github.com/snngrow/snngrow.git
    cd snngrow
    python setup.py install


Run the following command in your terminal to install the project:

Install the latest version from PyPI::

    pip install snngrow

Install from GitHub::

    git clone https://github.com/snngrow/snngrow.git
    cd snngrow
    python setup.py install
